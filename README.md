# EPI 553: Data Visualization in R

Interactive lecture slides for the Data Visualization module of **EPI 553, Principles of Statistical Inference II** at the University at Albany, SUNY (Spring 2026).

## View the Slides

[https://muntasirmasum.github.io/epi553-data-viz/](https://muntasirmasum.github.io/epi553-data-viz/)

## Topics Covered

- Why visualization matters (Anscombe's Quartet, the Datasaurus Dozen)
- The Grammar of Graphics and ggplot2's layered system
- Perception, pre-attentive attributes, and chart type selection
- Editorial styling and direct labeling (inspired by NYT/FiveThirtyEight)
- Iterative plot refinement from default to publication-ready
- Color theory, accessibility, and colorblind-safe palettes
- Interactive graphics with plotly and ggiraph
- Advanced geoms: raincloud plots, ridgelines, uncertainty visualization
- Multi-panel layouts with patchwork
- Annotation, spatial maps, summary tables with gt
- Visualizing regression model outputs (forest plots, predicted probabilities, marginal effects, diagnostics)
- The #30DayChartChallenge community

## Built With

- [Quarto](https://quarto.org/) RevealJS presentation format
- R and ggplot2 ecosystem
- Custom SCSS theme with Playfair Display and Atkinson Hyperlegible fonts
